#  AGROLENS

AGROLENS is a full-stack web application designed to help farmers and agricultural enthusiasts make informed crop decisions based on climate suitability, growing conditions, and crop intelligence.

It provides detailed information about crops commonly grown in West Africa, including recommended climate zones, growing requirements, pest risks, and farming recommendations.

---

##  Features

*  Crop intelligence database with 50+ crops
*  Climate zone suitability scoring
*  Detailed crop profiles
*  Water and sunlight requirements
*  Fertilizer recommendations
*  Best planting season information
*  Pest and disease risk alerts
*  Crop search and filtering
*  Responsive user interface

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/Chris01120/AGROLENS.git
```

### Backend

```bash
cd Backend/myproject/backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend

```bash
cd Frontend

npm install

npm run dev
```
